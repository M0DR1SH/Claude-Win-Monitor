Claude Win Monitor v1.9.1

Guide d'installation

Moniteur de quotas Claude en temps réel pour Windows. Affiche votre consommation de session (5 h glissantes), hebdomadaire (7 jours) et votre budget mensuel — directement sur votre bureau.
Interface disponible en 5 langues (FR, EN, DE, IT, ES) et multi-devises (€, $, ou tout symbole libre).

1

Prérequis

  • Windows 10 ou Windows 11 (64 bits)
  • Un navigateur compatible Chrome : Google Chrome, Microsoft Edge, Arc, Brave…
    (nécessaire pour l'extension automatique — voir section 7. Sans navigateur compatible, la clé de session peut être saisie manuellement.)
  • Un compte Claude.ai actif (abonnement Pro ou Team)
Aucune installation de Python ou d'autre logiciel n'est nécessaire.
L'application est autonome et embarque toutes ses dépendances (~59 Mo installés).
2

Contenu de l'archive

L'archive Claude-Win-Monitor-v1.9.1.zip contient :

00-README.txt ← démarrage rapide FR + EN 01-INSTALLATION.html ← ce guide 02-Installateur/ └─ Claude-Win-Monitor-Setup.exe ← programme d'installation 03-Extension-Chrome/ ├─ manifest.json ├─ background.js └─ icon.png SHA256SUMS.txt VERSION.txt
3

Avant d'installer — antivirus

L'application est compilée en binaire natif Windows (pas de script Python embarqué).

Le fichier d'installation est sûr : 0 détection sur 71 moteurs antivirus.
Vérification indépendante : VirusTotal — Claude-Win-Monitor-Setup.exe

Si vous disposez d'un antivirus, désactivez-le temporairement pendant l'installation, puis réactivez-le et ajoutez une exclusion sur le dossier d'installation (voir section 5).

4

Installation

4.1 Lancer l'installateur

Double-cliquez sur Claude-Win-Monitor-Setup.exe (~16 Mo).

En survolant le fichier, vous pouvez vérifier ses métadonnées :

4.2 Alerte SmartScreen (si elle apparaît)

Windows peut afficher un écran bleu "Windows a protégé votre ordinateur".

Cette alerte apparaît pour tout logiciel nouvellement publié et non signé numériquement. Elle ne signifie pas que le fichier est dangereux.
1
Cliquez sur Informations complémentaires
2
Cliquez sur Exécuter quand même

4.3 Étapes de l'assistant

Les zones en bleu indiquent des éléments optionnels. Les flèches rouges indiquent le bouton à cliquer.

Étape 1 — Dossier de destination

Dossier par défaut : C:\Program Files (x86)\Claude-Win-Monitor (~59 Mo). Modifiable via Parcourir. Cliquez sur Suivant.

Étape 2 — Dossier du menu Démarrer

Raccourci créé sous Claude Win Monitor. Cliquez sur Suivant.

Étape 3 — Prêt à installer

Vérifiez le récapitulatif et cliquez sur Installer.

Étape 4 — Fin de l'installation

Laissez Lancer Claude Win Monitor coché et cliquez sur Terminer.

5

Après l'installation — exclusion antivirus

Réactivez votre antivirus et ajoutez une exclusion permanente :

Chemin à exclure : C:\Program Files (x86)\Claude-Win-Monitor

Cherchez Exclusions ou Exceptions dans les paramètres de votre antivirus.

Claude Win Monitor apparaît dans Paramètres → Applications installées :

7

Installation de l'extension navigateur

Pourquoi cette extension ?

Pour afficher vos statistiques, Claude Win Monitor a besoin de vos identifiants de session — une clé temporaire générée par Claude.ai à chaque connexion. L'extension Claude Session Helper la récupère automatiquement depuis votre navigateur et la transmet à l'application, sans aucune intervention de votre part.

Si vous préférez ne pas installer d'extension, vous pouvez saisir la clé manuellement dans les Paramètres (voir section 9).

7.1 Ouvrir la gestion des extensions

Chrome / Edge / Brave : tapez chrome://extensions dans la barre d'adresse.
Arc : menu navigateur (icône en haut à gauche) → Extensions → Manage Extensions.

7.2 Activer le mode développeur et charger l'extension

1
Activez Mode développeur (bouton en haut à droite)
2
Cliquez sur Charger l'extension non empaquetée
3
Naviguez jusqu'au dossier dézippé, sélectionnez 03-Extension-Chrome et confirmez

7.3 Vérifier l'installation

L'extension Claude Session Helper apparaît dans la liste, activée.

7.4 Alerte périodique du navigateur

"Désactivez les extensions de développeur pour votre sécurité"
Cliquez sur Ignorer ou Ne plus afficher. L'extension continue de fonctionner normalement.
8

Premier lancement

8.1 Connexion automatique

Si l'extension est installée et que vous êtes connecté à claude.ai, la connexion s'effectue automatiquement en quelques secondes.

Le badge de l'en-tête passe au vert :

L'application affiche vos quotas en temps réel.

8.2 Connexion manuelle (sans extension)

Si la connexion automatique ne fonctionne pas, le badge indique une erreur :

Dans ce cas, saisissez la clé manuellement :

1
Cliquez sur Paramètres (barre du bas)
2
Dans la section Clé de Session, choisissez la Méthode manuelle
3
Dans votre navigateur, ouvrez claude.ai, puis faites un clic droit et choisissez Inspecter (ou appuyez sur F12)
4
Allez dans Application → Cookies → https://claude.ai → sessionKey
5
Copiez la valeur et collez-la dans le champ de l'application
6
Cliquez sur Sauvegarder & Relancer
9

Paramètres

Cliquez sur Paramètres pour ouvrir la fenêtre de configuration.

9.1 Clé de session

  • Méthode automatique (recommandée) — activez l'extension navigateur, elle gère tout automatiquement. Un lien Ouvrir le guide d'installation est disponible dans la fenêtre.

Choisissez € ou $, ou saisissez un symbole libre (1 à 3 caractères) — ex. : CHF, £, ¥.

9.4 Sauvegarder

Cliquez sur Sauvegarder & Relancer — l'application redémarre automatiquement avec les nouveaux paramètres.

10

Fenêtre Informations

Cliquez sur Informations pour afficher la version installée, la date de build et le lien vers l'auteur.

11

Icône dans la barre des tâches

Claude Win Monitor s'exécute en arrière-plan avec une icône dans la zone de notification (coin inférieur droit de l'écran).

11.1 Icône masquée ?

Windows peut masquer l'icône dans le sous-menu ^ des icônes cachées. Pour l'afficher en permanence :

Paramètres → Personnalisation → Barre des tâches → Autres icônes de barre d'état système
→ activez Moniteur de quotas Claude en temps réel

11.2 Utilisation de l'icône tray

  • Survol — affiche le pourcentage de session en cours sans ouvrir la fenêtre
  • Clic gauche — bascule la fenêtre au premier plan
  • Clic droit — menu contextuel : Afficher / Actualiser / Quitter

Mise à jour

Désinstallation

Votre configuration est conservée dans C:\Users\[votre nom]\AppData\Local\Claude-Win-Monitor\.
Supprimez ce dossier manuellement pour une désinstallation complète.

Claude Win Monitor v1.9.1

Installation guide

Real-time Claude quota monitor for Windows. Displays your session usage (5 rolling hours), weekly usage (7 days) and your monthly budget — directly on your desktop.
Interface available in 5 languages (FR, EN, DE, IT, ES) with multi-currency support (€, $, or any custom symbol).

1

Prerequisites

  • Windows 10 or Windows 11 (64-bit)
  • A Chrome-compatible browser: Google Chrome, Microsoft Edge, Arc, Brave…
    (required for the automatic extension — see section 7. Without a compatible browser, the session key can be entered manually.)
  • An active Claude.ai account (Pro or Team subscription)
No Python or other software installation is required.
The application is self-contained and includes all its dependencies (~59 MB installed).
2

Archive contents

The archive Claude-Win-Monitor-v1.9.1.zip contains:

00-README.txt ← quick start FR + EN 01-INSTALLATION.html ← this guide 02-Installateur/ └─ Claude-Win-Monitor-Setup.exe ← installer 03-Extension-Chrome/ ├─ manifest.json ├─ background.js └─ icon.png SHA256SUMS.txt VERSION.txt

Before installing — antivirus

The application is compiled as a native Windows binary (no embedded Python script).

The installer is safe: 0 detections out of 71 antivirus engines.
Independent verification: VirusTotal — Claude-Win-Monitor-Setup.exe

If you have an antivirus, temporarily disable it during installation, then re-enable it and add an exclusion for the installation folder (see section 5).

7

Browser extension installation

Why this extension?

To display your statistics, Claude Win Monitor needs your session credentials — a temporary key generated by Claude.ai at each login. The Claude Session Helper extension retrieves it automatically from your browser and passes it to the application, with no action required on your part.

If you prefer not to install an extension, you can enter the key manually in Settings (see section 9).

7.1 Open extension management

Chrome / Edge / Brave: type chrome://extensions in the address bar.
Arc: browser menu (icon top left) → Extensions → Manage Extensions.

7.2 Enable developer mode and load the extension

1
Enable Developer mode (toggle top right)
2
Click Load unpacked
3
Navigate to the unzipped folder, select 03-Extension-Chrome and confirm

7.3 Verify installation

The Claude Session Helper extension appears in the list, enabled.

7.4 Periodic browser alert

"Disable developer extensions for your security"
Click Dismiss or Don't show again. The extension continues to work normally.
8

First launch

8.1 Automatic connection

If the extension is installed and you are logged in to claude.ai, the connection happens automatically within a few seconds.

The header badge turns green:

The application displays your quotas in real time.

8.2 Manual connection (without extension)

If the automatic connection fails, the badge shows an error:

In that case, enter the key manually:

1
Click Settings (bottom bar)
2
In the Session Key section, choose the Manual method
3
In your browser, open claude.ai, then right-click and select Inspect (or press F12)
4
Go to Application → Cookies → https://claude.ai → sessionKey
5
Copy the value and paste it into the application field
6
Click Save & Restart
9

Settings

9.1 Session key

  • Automatic method (recommended) — activate the browser extension, it handles everything. An Open installation guide link is available in the window.
  • Manual method — follow the path F12 → Application → Cookies → https://claude.ai → SessionKey. An Open claude.ai link is available directly in the window.

9.2 Language

5 languages available — click the desired flag:

FR EN DE IT ES

9.3 Currency

Choose € or $, or enter a custom symbol (1 to 3 characters) — e.g.: CHF, £, ¥.

System tray icon

11.1 Icon hidden?

Settings → Personalization → Taskbar → Other system tray icons
→ enable Real-time Claude quota monitor

11.2 Using the tray icon

  • Hover — displays the current session percentage without opening the window
  • Left click — brings the window to the foreground
  • Right click — context menu: Show / Refresh / Quit
