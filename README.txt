================================================================================
  FR  INSTALLATION EN 3 ÉTAPES
  EN  INSTALLATION IN 3 STEPS
================================================================================

  FR  1. INSTALLER L'APPLICATION
         └─ Lancez "01-Installateur\Claude-Win-Monitor-Setup.exe"
            Si Windows affiche "Windows a protégé votre ordinateur" :
            → cliquez "Informations complémentaires" puis "Exécuter quand même"

  EN  1. INSTALL THE APPLICATION
         └─ Run "01-Installateur\Claude-Win-Monitor-Setup.exe"
            If Windows shows "Windows protected your PC":
            → click "More info" then "Run anyway"

  ──────────────────────────────────────────────────────────────────────────────

  FR  2. INSTALLER L'EXTENSION NAVIGATEUR (Chrome / Edge / Brave / Arc)
         └─ Dans votre navigateur, ouvrez : chrome://extensions
            Activez "Mode développeur" (en haut à droite)
            Cliquez "Charger l'extension non empaquetée"
            Sélectionnez le dossier "02-Extension-Chrome" et confirmez

  EN  2. INSTALL THE BROWSER EXTENSION (Chrome / Edge / Brave / Arc)
         └─ In your browser, open: chrome://extensions
            Enable "Developer mode" (top right)
            Click "Load unpacked"
            Select the "02-Extension-Chrome" folder and confirm

  ──────────────────────────────────────────────────────────────────────────────

  FR  3. LANCER L'APPLICATION
         └─ Ouvrez Claude Win Monitor depuis le Bureau ou le Menu Démarrer
            Ouvrez claude.ai dans votre navigateur
            → L'application se connecte automatiquement et affiche vos quotas

  EN  3. LAUNCH THE APPLICATION
         └─ Open Claude Win Monitor from the Desktop or Start Menu
            Open claude.ai in your browser
            → The app connects automatically and displays your quotas
